# 矩阵回顾

> 矩阵（Matrix）是数学中的基本概念，它是一个矩形阵列的数值或代数对象的集合。
> 在编程和算法中广泛使用，尤其是在线性代数、机器学习、图像处理等领域。

## 1.矩阵定义

```
 A = [
       a_{11}, a_{12}, ..., a_{1n},
       a_{21}, a_{22}, ..., a_{2n},
       ...
       a_{m1}, a_{m2}, ..., a_{mn}
     ]
```

## 2.转置矩阵

> 矩阵 A 的转置记为 (A^T) 或 (A')，其中 (A^T) 的元素 (a'{ij}) 与原矩阵 A 中的元素 (a{ji}) 相对应

```
 A^T = [
       a_{11}, a_{21}, ..., a_{m1},
       a_{12}, a_{22}, ..., a_{m2},
       ...
       a_{1n}, a_{2n}, ..., a_{mn}
     ]
```

## 3.矩阵加减法

```
B + C = [
       b_{11} + c_{11}, b_{12} + c_{12}, ..., b_{1n} + c_{1n},
       b_{21} + c_{21}, b_{22} + c_{22}, ..., b_{2n} + c_{2n},
       ...
       b_{m1} + c_{m1}, b_{m2} + c_{m2}, ..., b_{mn} + c_{mn}
     ]

```

## 4.矩阵乘法

> 矩阵乘法要求第一个矩阵的列数等于第二个矩阵的行数。
> 若 A 为 m×n 矩阵，B 为 n×p 矩阵，则它们的乘积 AB 是一个 m×p 矩阵

$$
    AB = [
       \sum_{k=1}^{n} a_{1k}b_{k1}, \sum_{k=1}^{n} a_{1k}b_{k2}, ..., \sum_{k=1}^{n} a_{1k}b_{kp},
       \sum_{k=1}^{n} a_{2k}b_{k1}, \sum_{k=1}^{n} a_{2k}b_{k2}, ..., \sum_{k=1}^{n} a_{2k}b_{kp},
       ...
       \sum_{k=1}^{n} a_{mk}b_{k1}, \sum_{k=1}^{n} a_{mk}b_{k2}, ..., \sum_{k=1}^{n} a_{mk}b_{kp}
     ]
$$

## 5.逆矩阵

> 矩阵 A 的逆矩阵记作 A^-1，如果 A 可逆，则 A^-1 是存在且唯一
